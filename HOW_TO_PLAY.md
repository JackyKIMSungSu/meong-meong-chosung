# 🐶 멍멍 초성게임 - 게임 방법

> 한글 단어의 **초성**(첫 자음)만 보고 정답을 맞히는 두뇌 게임입니다!
> 예시: `ㄱㅊㅉㄱ` → **김치찌개** 🍲

---

## 🚀 빠른 시작

1. 프로젝트 폴더에서 로컬 서버를 실행합니다:
   ```bash
   python3 -m http.server 8765
   ```
2. 브라우저에서 `http://localhost:8765` 접속
3. 모드 → 카테고리 → 난이도 선택 → **게임 시작 🚀**

---

## 🎮 게임 모드 (3종)

### 🎯 클래식 모드
- 무작위 **10문제**를 풀고 점수를 합산
- 골든 라운드(점수 2배)가 가끔 등장 ✨
- 정답률에 따라 등급 부여 (초성 마스터 ~ 입문자)

### 💀 서바이벌 모드
- **라이프 ❤️ 3개**로 시작, 무한 진행
- 오답·시간초과·패스 시 라이프 차감
- **5문제마다 제한 시간 1초씩 단축** (점점 빨라짐!)
- 라이프가 모두 소진되면 게임 종료

### ⚡ 스피드런 모드
- **60초** 동안 최대한 많은 문제를 맞히기
- 시간 압박 속에서 콤보를 쌓아보세요
- 정답 수 = 등급 (광속의 손 → 쾌속 플레이어 → 날렵한 손)

## 📂 카테고리 (6종)

| 이모지 | 카테고리 | 예시 단어 |
|:---:|:---|:---|
| 🍔 | 음식 | 김치찌개, 떡볶이, 짜장면 |
| 🐶 | 동물 | 강아지, 코끼리, 펭귄 |
| 🌏 | 나라 | 대한민국, 프랑스, 호주 |
| ⚽ | 스포츠 | 축구, 양궁, 컬링 |
| 🎬 | 영화 | 기생충, 부산행, 인터스텔라 |
| 🏠 | 생활 | 냉장고, 스마트폰, 이어폰 |

각 카테고리당 25개 이상의 단어가 준비되어 있습니다.

## 🔥 난이도

| 난이도 | 제한 시간 | 힌트 | 기본 점수 |
|:---:|:---:|:---:|:---:|
| 😊 쉬움 | 30초 | 3개 | 10점 |
| 😎 보통 | 20초 | 2개 | 15점 |
| 🥵 어려움 | 10초 | 1개 | 25점 |

---

## 💯 점수 시스템

```
획득 점수 = (기본 점수 + 시간 보너스 - 힌트 페널티 + 콤보 보너스) × 배수
```

| 항목 | 계산 |
|:---|:---|
| **기본 점수** | 난이도에 따라 10/15/25점 |
| **시간 보너스** | 남은 시간(초) 만큼 추가 |
| **힌트 페널티** | 힌트 1개 사용당 -3점 |
| **콤보 보너스** | 콤보 2 이상부터 `콤보 × 5점` |
| **더블 점수 💎** | 다음 정답에 ×2 |
| **골든 라운드 ✨** | 정답 시 ×2 |

> 💡 **콤보 + 더블 + 골든** 트리플 콤보 시 점수 폭발! 🎆

---

## 💡 힌트 시스템

힌트 버튼을 누르면 **3단계**로 정보가 공개됩니다:

1. **1단계**: 글자 수 알려주기 (예: "글자 수: 4자")
2. **2단계**: 의미 힌트 (예: "얼큰한 국물 요리")
3. **3단계**: 첫 글자 공개 (예: "첫 글자: '김'")

힌트를 사용할수록 점수에서 3점씩 차감됩니다.

---

## 🎁 아이템 시스템

정답을 맞히면 **25~55% 확률**로 아이템이 드롭됩니다 (콤보가 높을수록 드롭률 ↑).

| 아이템 | 효과 |
|:---:|:---|
| ⏱️ **시간 +5초** | 남은 시간을 5초 추가 |
| 👁️ **글자 공개** | 랜덤 글자 1개를 즉시 공개 |
| 💎 **더블 점수** | 다음 정답 점수 2배 |

화면 상단의 인벤토리 바에서 아이콘을 **클릭**하면 사용됩니다.

---

## 🔥 콤보 시스템

- **연속 정답** 시 콤보 카운터가 증가합니다
- 콤보 2 이상부터 보너스 점수 (`콤보 × 5점`)
- 콤보 3 이상에서는 더 화려한 컨페티 + 환호 효과음
- **오답·시간초과·패스** 시 콤보 초기화 ❌

---

## ✨ 골든 라운드

**클래식 모드**에서 15% 확률로 등장하는 특별 라운드:

- 화면 전체가 황금빛으로 빛남
- 마스코트가 흥분 (`황금이다 멍! ✨`)
- 정답 시 **획득 점수 2배** + 대형 컨페티
- 골든 정답 1회 시 **✨ 황금손** 업적 해금

---

## 🐶 마스코트 진화

점수에 따라 마스코트가 단계별로 변신합니다:

| 점수 | 진화 형태 |
|:---:|:---:|
| 0점~ | 🐶 강아지 |
| 100점~ | 🦊 여우 |
| 300점~ | 🐺 늑대 |
| 600점~ | 🦁 사자 |
| 1000점~ | 🐉 **드래곤** (최종) |

> 진화 순간 화려한 회전 애니메이션 + 전용 효과음이 재생됩니다!
> 드래곤까지 도달하면 **🐉 드래곤** 업적이 해금됩니다.

---

## 🎖️ 업적 (총 12종)

게임을 플레이하며 다양한 조건을 달성하면 업적이 해금됩니다.
업적 진행도는 시작 화면 하단의 **🏅 업적 보기** 버튼에서 확인할 수 있습니다.

| 업적 | 조건 |
|:---:|:---|
| 🎯 첫 정답 | 첫 정답을 맞히세요 |
| 🔥 연속 5콤보 | 5콤보 달성 |
| 🌋 연속 10콤보 | 10콤보 달성 |
| 💯 퍼펙트 | 클래식 모드에서 100% 정답 |
| 🧠 천재 | 힌트 없이 클래식 클리어 (퍼펙트 + 노힌트) |
| 💨 스피드스터 | 스피드런에서 15문제 이상 |
| 💪 생존자 | 서바이벌에서 20문제 이상 |
| ✨ 황금손 | 골든 라운드 정답 |
| 🎁 수집가 | 아이템을 5번 사용 |
| 🐉 드래곤 | 마스코트 최종 진화 도달 |
| ⭐ 500점 | 한 게임에서 500점 획득 |
| 🌟 1000점 | 한 게임에서 1000점 획득 |

> 업적 달성 시 화면 상단에 **금색 토스트 알림**이 등장합니다!

## 🏆 최고 기록

- 모드별 최고 점수가 **localStorage**에 자동 저장됩니다
- 시작 화면의 모드 버튼 옆에 베스트 점수 표시 🏆
- 신기록 갱신 시 결과 화면에 **🏆 신기록 달성!** 배너가 표시됩니다

> 브라우저 데이터를 초기화하면 기록이 사라지니 주의!

## 🎯 점수 부스팅 꿀팁

1. **빠르게 답하기** — 시간 보너스가 가장 큰 비중
2. **힌트는 최후의 수단** — 점수 페널티가 누적됨
3. **콤보 끊기지 않기** — 모르는 문제는 패스보다 신중히
4. **💎 더블 점수는 골든 라운드에 사용** — 점수 4배 폭발!
5. **👁️ 글자 공개는 어려운 문제에 활용** — 시간이 부족할 때 시간도 절약
6. **⏱️ 시간 +5초는 시간이 5초 이하일 때 사용** — 효과 극대화

## ⌨️ 조작 방법

| 동작 | 키 / 버튼 |
|:---|:---|
| 정답 입력 | 키보드로 직접 입력 |
| 정답 제출 | `Enter` 또는 ✏️ 버튼 |
| 힌트 사용 | 💡 힌트 버튼 |
| 패스 | ⏭ 패스 버튼 |
| 효과음 ON/OFF | 🔊 / 🔇 버튼 |
| 아이템 사용 | 인벤토리 아이템 아이콘 클릭 |
